\section{Introduction}
We aim to improve upon the classic filter wheel and monochromatic camera model. 
We simulate a system of captures which then reconstructs the XYZ color pallette with minimal error from a set of filter patches. 
Prior work has been limited by computational resources, and generally relies on heuristic measures to select filters.
We provide a minimal error reconstruction for the set of Rosco Supergel\cite{rosco_mycolor} filters, and compute the relationship between number of filters $N$ and minimum $\delta E$ under standard illuminants.

Further, we provide a framework for filter selection for any target spectral response function. 
By generalizing our target function, we allow for applications beyond RGB reconstruction, such as hyperspectral imaging and multispectral imaging, or even non-imaging applications such as spectral sensing.
